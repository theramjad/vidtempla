Common questions

Quick answers to the stuff people usually ask.

What exactly does VidTempla do?
It connects AI tools (like Claude or Cursor) to your YouTube channel. Instead of you manually updating descriptions, managing playlists, or replying to comments — your AI does it for you through VidTempla.
What's an MCP server?
It's a way for AI assistants to use tools. Think of it like giving your AI a set of buttons it can press — search videos, update descriptions, check analytics. You just add VidTempla as a server in your AI tool and it gets YouTube superpowers.
Is my YouTube channel safe?
Yes. You can create read-only API keys so your AI can look but not touch. When you do give write access, every single change is saved in version history so you can undo anything instantly.
Which AI tools work with VidTempla?
Any tool that supports MCP — that includes Claude (desktop and CLI), Cursor, Windsurf, and more. The REST API works with literally anything that can make web requests.
Do I need to know how to code?
Not at all. The MCP server works by just adding a config to your AI tool — copy, paste, done. The REST API is there for developers who want to build custom stuff, but it's totally optional.
Can I manage multiple YouTube channels?
Yes. Free gets you one channel, Pro gives you five, and Business is unlimited. All your channels show up in the same dashboard and the same AI connection.
What happens to my data?
VidTempla doesn't store your YouTube data. It talks to YouTube's servers in real-time whenever your AI needs something. The only things we store are your templates and description history.
Is it really open source?
Yep, 100%. The code is on GitHub. You can read it, fork it, or self-host it. We make money from the hosted version — the one you don't have to set up yourself.
