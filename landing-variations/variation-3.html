FAQ

Common questions

What is VidTempla?

It's a bridge between AI tools (like Claude or Cursor) and your YouTube channel. You connect your account, give your AI an API key, and it can manage your channel for you.

What's an MCP server?

It's a way for AI tools to directly use YouTube features — like searching, editing playlists, or reading comments — without you writing any code.

Is my YouTube account safe?

Yes. You choose exactly what each API key can do. You can make keys read-only so the AI can look but not change anything. You can also revoke access any time.

What if the AI messes something up?

Every change to your video descriptions is saved. You can undo any edit with one click and go back to any previous version.

Which AI tools does this work with?

Anything that supports MCP — Claude Desktop, Cursor, Windsurf, and more. The REST API works with any tool or script that can make web requests.

Can I manage multiple channels?

Yes. The free plan supports one channel. Pro gives you three, and Business is unlimited.

Is it open source?

Yes, fully open source on GitHub. You can see exactly how it works, contribute, or even self-host it.

Do I need to know how to code?

Not at all. If you can use an AI tool like Claude, you already have everything you need. The AI handles the technical parts.
